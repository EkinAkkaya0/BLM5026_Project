# Kurulum Yönergeleri
## 1. Depoyu Klonla

Terminal veya PowerShell açın ve aşağıdaki komutu çalıştırın:

git clone https://github.com/EkinAkkaya0/BLM5026_Project.git

## 2. Unity ile Aç

Unity Hub’ı açın.

Open > Add project from disk seçeneğine tıklayın.

Klonladığınız klasörü seçin:

BLM5026_Project/

## 3. Sahneyi Çalıştır

Unity Editor açıldığında Scenes klasöründeki ana sahneyi yükleyin.

Play butonuna basarak oyunu başlatın.
